Getting Started with Vercel Web Analytics

Learn how to enable analytics tracking and view visitor data

Overview

This guide will help you get started with using Vercel Web Analytics on your project. Vercel Web Analytics provides insights into your website's visitor behavior, helping you understand how users interact with your site.

We'll walk you through enabling analytics, adding the package to your project, deploying to Vercel, and viewing your data in the dashboard.

Install Vercel CLI

Install the Vercel CLI using your preferred package manager:

Using pnpm:

pnpm i vercel

Using yarn:

yarn add vercel

Using npm:

npm install vercel

Using bun:

bun add vercel

Enable Web Analytics in Vercel

  1. 1. Navigate to the Vercel dashboard
  2. 2. Select your project
  3. 3. Click the Analytics tab
  4. 4. Click Enable in the dialog

ðŸ’¡ Note: Enabling Web Analytics will add new routes (scoped at /_vercel/insights/*) after your next deployment.

Add @vercel/analytics to Your Project

Using the package manager of your choice, add the @vercel/analytics package:

Using pnpm:

pnpm add @vercel/analytics

Using yarn:

yarn add @vercel/analytics

Using npm:

npm install @vercel/analytics

Using bun:

bun add @vercel/analytics

Implementation for Plain HTML Sites

Deploy Your App to Vercel

Deploy your app using the following command: 

vercel deploy

Alternatively, we recommend connecting your project's Git repository, which will automatically deploy your latest commits to the main branch without requiring terminal commands.

ðŸ’¡ Tip: Once your app is deployed, it will automatically start tracking visitors and page views.

If everything is set up properly, you should be able to see a Fetch/XHR request in your browser's Network tab from /_vercel/insights/view when you visit any page.

View Your Data in the Dashboard

  1. 1. Once your app is deployed and users have visited your site, navigate to your Vercel dashboard
  2. 2. Select your project
  3. 3. Click the Analytics tab
  4. 4. After a few days of visitors, you'll start seeing data in the analytics dashboard
