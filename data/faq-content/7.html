Currently, updating is done via reinstallation from the software store.
It is also possible to try updating directly from the software store or using the command flatpak update, but this method has not been tested. If the update fails, we recommend following the reinstallation process indicated below.

1. Uninstall if you have an old version (Recommended)
2. Install Trinity Launcher from the software store

Command to uninstall the old version:
flatpak uninstall com.trench.trinity.launcher
