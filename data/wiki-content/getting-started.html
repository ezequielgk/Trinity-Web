Welcome to Trinity Launcher

What is Trinity Launcher?

Trinity Launcher is an open source solution designed specifically to allow Linux users to run Minecraft Bedrock Edition natively and efficiently.

Developed as an open source project under the BSD License, Trinity Launcher uses Flatpak technology to provide a secure, isolated, and easy-to-manage installation on any Linux distribution.

Key Features

  • Open Source and completely free
  • Compatible with all Linux distributions
  • Secure installation via Flatpak
  • Support for multiple Minecraft versions

System Requirements

  • Linux (any distribution with Flatpak)
  • 2GB RAM minimum (8GB recommended)
  • OpenGL 3.0 or Vulkan compatible GPU
  • 2GB free disk space
  • Minecraft Bedrock x86-64 APK

Why Choose Trinity?

Designed for Linux

Specifically optimized for the Linux ecosystem, without the need for emulation layers.

Secure & Isolated

Uses Flatpak sandbox technology to keep the game isolated from your main system.

Complete Experience

Manage all versions, mods, texture packs and resources from a unified interface.

Open Source

Transparent code under BSD License, developed by the community for the community.
