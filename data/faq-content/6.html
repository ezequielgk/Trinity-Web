Este es un problema común relacionado con GPUs dedicadas Nvidia o Amd Viejas. La solución es usar variables de entorno para forzar el uso de una GPU específica:

Solución 1 - GPU integrada:
DRI_PRIME=0 flatpak run com.trench.trinity.launcher

Solución 2 - GPU dedicada:
DRI_PRIME=1 flatpak run com.trench.trinity.launcher

¿Cuál usar?
• Prueba primero con DRI_PRIME=0
• Si no funciona, prueba con DRI_PRIME=1

⚠️ Nota: Este problema suele ocurrir en sistemas con GPUs dedicadas Amd o Nvidia viejas. La solución fuerza el uso de un renderizador específico.
