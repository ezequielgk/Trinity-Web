Instalación de Trinity Launcher

Guía completa para instalar Trinity Launcher en tu sistema Linux

Antes de empezar

Trinity Launcher se distribuye como un paquete Flatpak para garantizar compatibilidad y seguridad en todas las distribuciones de Linux. Asegúrate de tener Flatpak instalado antes de continuar.

Paso 1: Instalar Flatpak

Para instalar Flatpak en tu distribución de Linux, visita la página oficial y busca las instrucciones específicas para tu sistema:

Ir a Flatpak.org

Una vez que tengas Flatpak instalado y configurado, puedes continuar con la instalación de Trinity Launcher.

Paso 2: Instalar Trinity Launcher

Método A: Instalación desde Repositorio Testing-Beta

MÁS ACTUALIZADO

Este método instala Trinity Launcher directamente desde el repositorio testing-beta, obteniendo las versiones más recientes y actualizaciones automáticas.

Ejecuta estos comandos en orden en tu terminal:

1. Agregar el repositorio Trinity:

flatpak remote-add --if-not-exists trinity https://trinity-flatpak.codeberg.page/com.trench.trinity.launcher.flatpakrepo

2. Instalar el runtime de KDE (Qt6):

flatpak install flathub org.kde.Platform//6.9

3. Instalar la base de Qt WebEngine para Qt6:

flatpak install flathub io.qt.qtwebengine.BaseApp//6.9

4. Instalar Trinity Launcher:

flatpak install trinity com.trench.trinity.launcher

Ventajas de este método: Acceso directo a las últimas versiones beta, actualizaciones automáticas cuando estén disponibles, y instalación simplificada en cuatro comandos.

Método B: Instalación desde tienda de software

Primero debes agregar el repositorio Trinity ejecutando este comando en tu terminal:

flatpak remote-add --if-not-exists trinity https://trinity-flatpak.codeberg.page/com.trench.trinity.launcher.flatpakrepo

Una vez agregado el repositorio, podrás buscar e instalar Trinity Launcher directamente desde tu tienda de software (GNOME Software, KDE Discover, etc.) buscando "Trinity Launcher".

Nota: Es posible que necesites reiniciar tu tienda de software después de agregar el repositorio para que Trinity Launcher aparezca en los resultados de búsqueda.

Paso 3: Ejecutar Trinity Launcher

Una vez instalado, puedes ejecutar Trinity Launcher de las siguientes maneras:

Desde el menú de aplicaciones: Busca "Trinity Launcher" en tu lanzador de aplicaciones

Desde terminal:

flatpak run com.trench.trinity.launcher

Problemas de instalación

Si encuentras problemas durante la instalación, verifica que tienes los repositorios de Flatpak configurados correctamente y reinicia tu sesión después de instalar Flatpak por primera vez.
