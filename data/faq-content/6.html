This is a common issue related to old dedicated Nvidia or AMD GPUs. The solution is to use environment variables to force the use of a specific GPU:

Solution 1 - Integrated GPU:
DRI_PRIME=0 flatpak run com.trench.trinity.launcher

Solution 2 - Dedicated GPU:
DRI_PRIME=1 flatpak run com.trench.trinity.launcher

Which one to use?
• Try first with DRI_PRIME=0
• If that doesn't work, try with DRI_PRIME=1

⚠️ Note: This issue usually occurs on systems with old dedicated AMD or Nvidia GPUs. The solution forces the use of a specific renderer.
