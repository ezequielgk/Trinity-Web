Trinity Launcher

The open source launcher for Minecraft Bedrock on Linux

What is Trinity Launcher?

Trinity Launcher is an open source solution designed specifically to allow Linux users to run Minecraft Bedrock Edition natively and efficiently.

Developed as an open source project under the BSD License, Trinity Launcher uses Flatpak technology to provide a secure, isolated, and easy-to-manage installation on any Linux distribution.
