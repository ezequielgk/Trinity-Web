Trinity Launcher Installation

Complete guide to installing Trinity Launcher on your Linux system

Before you start

Trinity Launcher is distributed as a Flatpak package to ensure compatibility and security across all Linux distributions. Make sure you have Flatpak installed before continuing.

Step 1: Install Flatpak

To install Flatpak on your Linux distribution, visit the official page and look for specific instructions for your system:

Go to Flatpak.org

Once you have Flatpak installed and configured, you can proceed with the Trinity Launcher installation.

Step 2: Install Trinity Launcher

Method A: Installation from Testing-Beta Repository

MOST UP-TO-DATE

This method installs Trinity Launcher directly from the testing-beta repository, getting the latest versions and automatic updates.

Run these commands in order in your terminal:

1. Add the Trinity repository:

flatpak remote-add --if-not-exists trinity https://trinity-flatpak.codeberg.page/com.trench.trinity.launcher.flatpakrepo

2. Install the KDE runtime (Qt6):

flatpak install flathub org.kde.Platform//6.9

3. Install the Qt WebEngine base for Qt6:

flatpak install flathub io.qt.qtwebengine.BaseApp//6.9

4. Install Trinity Launcher:

flatpak install com.trench.trinity.launcher

Advantages of this method: Direct access to the latest beta versions, automatic updates when available, and simplified installation in four commands.

Method B: Installation from software store

First you must add the Trinity repository by running this command in your terminal:

flatpak remote-add --if-not-exists trinity https://trinity-flatpak.codeberg.page/com.trench.trinity.launcher.flatpakrepo

Once the repository is added, you can search for and install Trinity Launcher directly from your software store (GNOME Software, KDE Discover, etc.) by searching for "Trinity Launcher".

Note: You may need to restart your software store after adding the repository for Trinity Launcher to appear in search results.

Step 3: Run Trinity Launcher

Once installed, you can run Trinity Launcher in the following ways:

From the applications menu: Search for "Trinity Launcher" in your application launcher

From terminal:

flatpak run com.trench.trinity.launcher

Installation issues

If you encounter problems during installation, verify that you have the Flatpak repositories configured correctly and restart your session after installing Flatpak for the first time.
