Actualmente, la actualización se realiza mediante reinstalación desde la tienda de software.
También es posible intentar actualizar directamente desde la tienda de software o utilizando el comando flatpak update, pero este método no ha sido probado. Si la actualización falla, recomendamos seguir el proceso de reinstalación indicado a continuación.

1. Desinstala si tienes una versión vieja (Recomendado)
2. Instala Trinity Launcher desde la tienda de software

Comando para desinstalar la versión vieja:
flatpak uninstall com.trench.trinity.launcher
